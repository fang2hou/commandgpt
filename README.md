# commandgpt

A TUI program to help you find commands using OpenAI models.

Currently, `GPT-3.5-Turbo` and `GPT-4` are supported.

## Preview

https://github.com/fang2hou/commandgpt/assets/6694722/8a98f426-ecdc-4f62-85a8-e12752afae30

## Usage

```shell
cg
```

## Install

### From source

go 1.20 is required.

```shell
go build -o cg .
mv cg /usr/local/bin
```
